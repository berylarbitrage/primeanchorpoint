# 工作许可验证指南 Work Authorization Verification Guide

本文档定义了每种身份类别所需的**最少验证材料**、对应的**税表**，以及**实体（公司）**的处理方式。

---

## 一、个人 Individual

### 1. 美国公民 US Citizen

| 项目 | 内容 |
|------|------|
| **Work Permit 验证材料（任选一）** | ① 美国护照 US Passport ② 美国出生证明 Birth Certificate ③ 入籍证书 Naturalization Certificate (N-550/N-570) ④ 公民身份证书 Certificate of Citizenship (N-560/N-561) |
| **最少材料** | **1 份**，以上任意一项即可 |
| **税表** | **W-9** |
| **额外税务文件** | 无 |

---

### 2. 绿卡持有人 Green Card (Permanent Resident)

| 项目 | 内容 |
|------|------|
| **Work Permit 验证材料（任选一）** | ① 绿卡 Green Card (I-551) ② 外国护照 + I-551 临时章 (Temporary I-551 stamp) |
| **最少材料** | **1 份**，以上任意一项即可 |
| **税表** | **W-9** |
| **额外税务文件** | 无 |

---

### 3. H-1B 签证持有人

| 项目 | 内容 |
|------|------|
| **Work Permit 验证材料** | I-797 Approval Notice（显示雇主、有效期） |
| **最少材料** | **1 份** — I-797 即可 |
| **Work Permit Verification 表单填写** | Document Type: `I-797`，Document Number: Receipt Number (如 EAC/SRC...)，Issue/Expiry: I-797 上的有效期，Category: `H-1B` |
| **税表（视 SPT 结果）** | 居民外国人 (SPT ≥ 183) → **W-9** · 非居民外国人 (NRA) 无条约 → **W-8BEN** · NRA + 主张条约豁免 → **Form 8233** |
| **NRA 额外税务文件** | 护照复印件 + 签证复印件 + I-94 |

---

### 4. L-1 签证持有人 (跨国公司内部调动)

| **最少材料** | **1 份** — I-797 |
| **Category** | `L-1` |
| **税表** | 同 H-1B 逻辑（视 SPT） |

### 5. O-1 签证持有人 (杰出人才)

| **Category** | `O-1` |
| **税表** | 同 H-1B 逻辑（视 SPT） |

### 6. TN 签证持有人 (USMCA/北美自由贸易)

| 项目 | 内容 |
|------|------|
| **Work Permit 验证材料（任选一）** | ① I-797 Approval Notice ② I-94 显示 TN 身份 |
| **最少材料** | **1 份** |
| **Category** | `TN` |
| **税表** | 视 SPT。注意：加拿大/墨西哥公民可能主张条约豁免 → Form 8233 |

### 7. E-1 / E-2 / E-3 签证持有人 (条约投资者/贸易商/澳洲专业人士)

|------|------|

| **最少材料** | **1 份** (优先 I-797) |
| **Category** | `E-1` / `E-2` / `E-3` |

### 8. R-1 / P-1 签证持有人 (宗教工作者/运动员艺术家)

| 项目 | 内容 |
|------|------|

| **最少材料** | **1 份** — I-797 |
| **Category** | `R-1` / `P-1` |
| **税表** | 视 SPT |

---

### 9. F-1 OPT (Optional Practical Training) — EAD 持有人

| 项目 | 内容 |
|------|------|
| **Work Permit 验证材料** | EAD 卡 (I-766)，卡上类别为 C03A (STEM) 或 C03B (非STEM) |
| **最少材料** | **1 份** — EAD 卡即可 |
| **Work Permit Verification 表单填写** | Document Type: `EAD Card (I-766)`，Document Number: 卡号，Issue/Expiry: 卡面日期，Category: `F-1 OPT` 或 `F-1 OPT STEM` |
| **税表** | F-1 学生前 5 个日历年为 Exempt Individual，不计入 SPT → 通常为 **W-8BEN** · 若主张中美税收条约 Article 20 → **Form 8233** · 超过 5 年且 SPT ≥ 183 → **W-9** |
| **NRA 额外税务文件** | 护照复印件 + 签证复印件 + I-94 |

---

### 10. F-1 CPT (Curricular Practical Training)

| 项目 | 内容 |
|------|------|
| **Work Permit 验证材料** | I-20（第 2 页有 CPT 授权批注，显示雇主和有效期） |
| **最少材料** | **1 份** — 带 CPT 批注的 I-20 |
| **Category** | `F-1 CPT` |
| **税表** | 同 F-1 OPT 逻辑 |

---

### 11. J-1 签证持有人 (交流访问者)

| 项目 | 内容 |
|------|------|
| **Work Permit 验证材料** | DS-2019（显示项目日期和授权信息） |
| **最少材料** | **1 份** — DS-2019 |
| **Category** | `J-1` |
| **税表** | J-1 非学生前 2 个日历年为 Exempt Individual → 通常 **W-8BEN** · 主张条约 → **Form 8233** · 超过豁免期且 SPT ≥ 183 → **W-9** |
| **NRA 额外税务文件** | 护照复印件 + 签证复印件 + I-94 |

### 12. EAD 持有人 — 各类别通用 (庇护 C08 / 婚姻 / H-4 C26 / DACA C33 / 其他)

以下 EAD 类别的验证方式完全相同，因为 EAD 卡本身就是独立的工作许可证明：

| EAD 类别代码 | 说明 |
|-------------|------|
| C08 | 庇护申请人 Asylum Applicant |
| A05 | 直系亲属/丧偶 Immediate Relative |
| C09 | 多元化签证 Diversity Visa |
| C10 | 家庭担保 Family-Sponsored |
| A10 | 其他直系亲属 |
| C26 | H-4 配偶 Dependent |
| A18 | 家庭就业优先 |
| C33 | DACA 童年入境暂缓遣返 |
| A12 | 多元化签证 |
| C03A | OPT STEM (见上方 F-1 OPT 部分) |
| C03B | OPT 非STEM (见上方 F-1 OPT 部分) |

| 项目 | 内容 |
|------|------|
| **Work Permit 验证材料** | EAD 卡 (I-766) |
| **最少材料** | **1 份** — EAD 卡即可（卡面包含类别代码、有效期、照片） |
| **Work Permit Verification 表单填写** | Document Type: `EAD Card (I-766)`，Document Number: 卡号 (SRC/EAC...)，Issue/Expiry: 卡面日期，Category: 对应的 EAD 类别 |
| **税表** | 大多数 EAD 持有人长期居住美国，SPT ≥ 183 → **W-9** · 如果是 NRA → **W-8BEN** · 主张条约 → **Form 8233** |

## 二、实体 Entity (公司/合伙企业)

### 1. 美国实体 US Entity

| 项目 | 内容 |
|------|------|
| **验证材料（任选一）** | ① EIN 确认信 (IRS CP 575/147C) ② 公司注册证书 Certificate of Incorporation / Formation ③ 州务卿注册文件 Secretary of State filing |
| **最少材料** | **1 份** |
| **税表** | **W-9** |

---

### 2. 外国实体 Foreign Entity

| 项目 | 内容 |
|------|------|
| **验证材料** | ① 公司注册/成立证书 Certificate of Incorporation ② 签字授权文件（证明签字人有权代表公司） |
| **最少材料** | **2 份** — 公司证明 + 授权文件 |
| **税表** | **W-8BEN-E** |
| **额外税务文件** | 如主张条约 → 条约相关支持文件 |

## 三、税表决策流程图 Tax Form Decision Tree

```
申请人类型？
├── 个人 Individual
│   ├── 美国公民？ → W-9
│   ├── 绿卡持有人？ → W-9
│   ├── 非公民非绿卡 →
│   │   ├── 计算 SPT (Substantial Presence Test)
│   │   │   ├── F/J/M/Q 签证？→ 豁免天数不计入 SPT
│   │   │   ├── 当年在美 ≥ 31天 且 加权天数 ≥ 183？
│   │   │   │   ├── 是 → Likely Resident Alien → W-9
│   │   │   │   └── 否 → Non-Resident Alien (NRA) ↓
│   │   │   └── NRA:
│   │   │       ├── 不主张条约？ → W-8BEN
│   │   │       │   需要：护照 + 签证 + I-94
│   │   │       └── 主张条约且为个人劳务？ → Form 8233
│   │   │           需要：护照 + 签证 + I-94
│   │   │           (其他收入如利息版税等仍在 W-8BEN 中注明条约条款)
│   │   └── 注意：Form 8233 需要人工审核确认
│   │
└── 实体 Entity
    ├── 美国实体？ → W-9
    └── 外国实体？ → W-8BEN-E
        └── 如主张条约 → 附条约支持文件
```

## 四、Work Permit Verification 表单填写对照表

系统中 Work Permit Verification 弹窗需要填写以下字段：

| 字段 | 说明 |
|------|------|
| Document Type | 证件类型（见下方对照） |
| Document Number | 证件编号 |
| Issue Date | 签发日期 |
| Expiry Date | 到期日期（公民/绿卡可留空或填 N/A） |
| USCIS Category | 签证/EAD 类别 |
| Notes | 审核备注 |

### Document Type 填写对照：

| 身份类别 | Document Type | Document Number 示例 |
|----------|--------------|---------------------|
| US Citizen (护照) | US Passport | 护照号码 |
| US Citizen (出生证明) | Birth Certificate | 证书编号 |
| US Citizen (入籍) | Naturalization Certificate | N-550 编号 |
| Green Card | Permanent Resident Card (I-551) | 卡号 (3位字母+10位数字) |
| H-1B / L-1 / O-1 / TN / E / R-1 / P-1 | I-797 Approval Notice | Receipt Number (如 EAC2190012345) |
| F-1 OPT | EAD Card (I-766) | 卡号 |
| F-1 CPT | I-20 (CPT Authorization) | SEVIS ID (N00...) |
| J-1 | DS-2019 | Program Number |
| 各类 EAD (C08/C26/C33 等) | EAD Card (I-766) | 卡号 |

## 六、NRA 税务附加文件清单

当确定为非居民外国人 (NRA) 需要 W-8BEN 或 Form 8233 时，额外需要：

| 文件 | 用途 | 必须？ |
|------|------|--------|
| 护照复印件 Passport Copy | 身份和国籍证明 | 是 |
| 签证复印件 Visa Copy | 签证类别和有效期 | 是 |
| I-94 入境记录 Arrival/Departure Record | 在美天数和身份状态 | 是 |
| 工作许可文件 Work Auth Docs | 已在 Work Permit Verification 中收集 | 已收集 |

如主张条约豁免 (Form 8233)，还需要：
- 条约相关支持文件（系统会在税务问卷中标记需人工审核）

## 七、注意事项

1. **I-9 是独立流程** — 本文档仅涉及 Work Permit Verification 和税表。I-9 (Employment Eligibility Verification) 有自己的 List A / List B+C 文件要求，由系统单独处理。

2. **SPT 计算由系统自动完成** — 税务居民问卷 (Tax Residency Questionnaire) 会自动计算加权天数并推荐税表。

3. **EAD 卡是最简单的验证** — 对于所有 EAD 持有人，一张有效的 EAD 卡即包含所需全部信息（照片、类别、有效期、卡号）。

4. **Form 8233 需要人工审核** — 系统会自动标记需要人工复核的情况。

5. **到期日管理** — 系统跟踪所有工作许可的到期日期，需及时续签。
